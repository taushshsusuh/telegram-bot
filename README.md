# بوت بحث breach.vip لـ Telegram

بوت Telegram يسمح للمستخدمين بالبحث عن أسماء المستخدمين في موقع breach.vip وتصفية النتائج لعرض معلومات Instagram فقط، بما في ذلك البريد الإلكتروني واسم المستخدم المرتبط.

## الميزات

*   البحث عن طريق اسم المستخدم في breach.vip.
*   تصفية النتائج لعرض بيانات Instagram فقط.
*   استخراج البريد الإلكتروني واسم المستخدم من نتائج Instagram.
*   استجابات واضحة ومنسقة في Telegram.

## الإعداد والتشغيل

### المتطلبات المسبقة

*   حساب Telegram وبوت API Token.
*   Docker (للتشغيل في بيئة حاويات).
